Detailed Description

Configuration structure for loading an image from pixel data.

This structure defines the configuration for loading an image from pixel data. It includes the pixel data, dimensions, alpha mask, and callback function.

Note
The pixel data format depends on the display type:
  • DISP_DEFAULT: 16-bit RGB565 format
  • DISP_PCD8544/SSD1306/U8G2: 8-bit monochrome format
