Detailed Description

Widget de checkbox interativo que permite alternar entre estados marcado e desmarcado.

Esta classe herda de WidgetBase e fornece funcionalidade completa para criar e gerenciar checkboxes interativos na tela. O CheckBox desenha um quadrado com bordas arredondadas e uma marca de verificação quando está no estado marcado. O widget pode ser configurado com diferentes tamanhos, cores, pesos de linha para a marca e callbacks para responder a mudanças de estado. O checkbox é totalmente funcional com suporte a toque em displays capacitivos e touchscreen.

Detecta se o CheckBox foi tocado pelo usuário.

Parameters
_xTouchPonteiro para a coordenada X do toque na tela.
_yTouchPonteiro para a coordenada Y do toque na tela.
Returns
True se o toque está dentro da área do CheckBox, False caso contrário.

Este método realiza múltiplas validações antes de processar o toque:

  • Verifica se o widget está visível, inicializado, carregado e habilitado
  • Valida que o teclado virtual não está ativo
  • Verifica que o checkbox está na tela atual
  • Aplica debounce para evitar múltiplos cliques
  • Verifica se o widget não está bloqueado
  • Usa tolerância de 2 pixels para melhorar a experiência do usuário Se todas as validações passarem e o toque estiver dentro da área do checkbox, o estado é alternado e o widget é marcado para redesenho.

◆ redraw()

void CheckBox::redraw ( )
overridevirtual

Redesenha o CheckBox na tela, atualizando sua aparência baseada em seu estado.

Este método é responsável por renderizar o checkbox na tela:

  • Verifica todas as condições necessárias para o redesenho
  • Desenha um retângulo arredondado com a cor apropriada (marcado ou desmarcado)
  • Desenha a borda do checkbox se m_borderWidth > 0
  • Desenha a marca de verificação (checkmark) se o checkbox está marcado
  • Usa a biblioteca Arduino_GFX_Library para o desenho Apenas redesenha se o checkbox está visível, inicializado, carregado, na tela atual e a flag m_shouldRedraw está configurada como true.
