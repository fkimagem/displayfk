Detailed Description

Widget de LED com tamanho e cor personalizáveis.

Esta classe herda de WidgetBase e fornece funcionalidade completa para criar e gerenciar LEDs interativos na tela. O Led desenha um círculo com efeito de brilho quando ligado, usando gradientes de cor para criar um efeito visual realista. O widget pode ser configurado com diferentes raios, cores para estado ligado/desligado e suporta modo claro/escuro. O LED é totalmente funcional com suporte a estados ligado/desligado e atualização visual.

◆ redraw()

void Led::redraw ( )
overridevirtual

Redesenha o LED na tela, atualizando sua aparência.

Exibe o LED com representação visual adequada baseada no estado atual:

  • Estado ligado: desenha múltiplos círculos concêntricos com gradiente de cor para criar efeito de brilho
  • Estado desligado: desenha círculo sólido com cor de fundo Apenas redesenha se o LED está visível, inicializado, carregado, na tela atual e a flag m_shouldRedraw está configurada como true.

Configura o widget LED com parâmetros definidos em uma estrutura de configuração.

Parameters
