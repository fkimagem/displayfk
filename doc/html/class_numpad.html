Detailed Description

Widget de teclado numérico que permite entrada de números através de um teclado na tela.

Esta classe herda de WidgetBase e fornece funcionalidade completa para criar e gerenciar teclados numéricos interativos na tela. O Numpad desenha um teclado numérico com teclas para dígitos (0-9), ponto decimal, operações matemáticas (incremento, decremento, inversão) e ações de controle (Delete, OK). O widget pode ser configurado com diferentes cores e fontes e trabalha em conjunto com NumberBox para entrada de dados numéricos. O teclado é totalmente funcional com suporte a detecção de toque e atualização de valores em tempo real.
