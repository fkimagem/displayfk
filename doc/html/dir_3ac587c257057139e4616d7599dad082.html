FT6X36 (FT6236/FT6336/FT6436L/FT6436)

This is a library for self-capacitive touch panel controllers produced by FocalTech Systems.

ATTENTION!

Only basic functionality is currently implemented. Has been tested only on ESP32 using a 2.7" e-ink display (GDEW027W3-T). @section autotoc_md10 Annotation The <a href="https://github.com/adafruit/Adafruit_FT6206_Library" >Adafruit_FT6206_Library was used as a base for the FT6X36 library.
