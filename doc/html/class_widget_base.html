Detailed Description

Classe base abstrata para todos os widgets.

Esta classe fornece funcionalidade comum para todos os widgets no sistema DisplayFK. Define métodos virtuais puros que devem ser implementados pelas classes derivadas, além de fornecer funcionalidades estáticas compartilhadas como gerenciamento de cores, callbacks, fontes e referências ao display. Esta classe não pode ser instanciada diretamente e deve ser herdada por classes de widgets concretos como Button, TextBox, Gauge, etc. Todos os widgets herdam de WidgetBase e implementam seus métodos virtuais puros.
